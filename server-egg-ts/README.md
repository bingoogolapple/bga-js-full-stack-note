# server-egg-ts

## 初始化项目

- 初始化 node 项目

```shell
mkdir server-egg-ts
cd server-egg-ts
npm init egg --type=ts
yarn
yarn autod
```

- 启动

```bash
yarn
yarn dev
open http://localhost:7001/
```

- 部署

```bash
yarn tsc
yarn start
```

- 清除编译生成的 js 文件

```shell
yarn clean
```

- 单测

```shell
yarn test
```

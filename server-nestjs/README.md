# server-nestjs

- 初始化项目

```bash
npm i -g @nestjs/cli
nest new server-nestjs
```

- 本地开发

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

- 单元测试

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
